<h1>About</h1>
<p>Laravel simple multi-tenancy (One Database)</p>

<h1>Deployment</h1>
<ul>
    <li>Run <code>composer install && composer update</code> </li>
    <li>copy .env.example as .env and update your .env file</li>
    <li>Run <code>php artisan project:install</code></li>
</ul>
